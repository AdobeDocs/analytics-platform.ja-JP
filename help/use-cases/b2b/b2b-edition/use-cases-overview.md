# B2B editionのユースケースの概要

このセクションでは、より複雑なB2Bの購入ジャーニーにおけるインサイトを提供することを目的とした、B2Bの特定のユースケースについて詳しく説明します。 B2Bの一般的な購入には、購買グループ、さまざまな関係者によるクロスチャネルの顧客接点、セールスサイクルのさまざまな段階での機会などが含まれます。


## 主要ペルソナ

Customer Journey Analytics B2B editionなら、次のようなペルソナを活用してデータを分析できます。

| ペルソナ | B2B分析の要件 |
|---|---|
| マーケティングマネージャー | キャンペーンのパフォーマンス、チャネルのパフォーマンスとライフサイクル、マーケティング ROI、リードジェネレーション、顧客維持、ブランドパフォーマンス |
| ビジネスアナリスト | 詳細なデータ、レポート作成時間の短縮、多くの関係者をサポートする柔軟性、インサイトの容易な提供。 |
| アカウント責任者 | 売上向上、機会の進行、顧客満足度、顧客獲得、主要インサイトへの迅速なアクセス、モバイルレポート。 |
| プロダクトマネージャー | 顧客満足度、製品の使用傾向、パフォーマンス傾向、収益拡大 |
| CMOまたはCEO | 売上向上、ブランドパフォーマンス、顧客エンゲージメント、市場拡大 |
| マーケティングオペレーションマネージャー | 効率性、データ精度、キャンペーンパフォーマンス、テクノロジー統合 |
| セールスオペレーションマネージャー | 効率性、データ精度、セールスパフォーマンス、商談の進行、テクノロジー統合 |


## ユースケース

上記のペルソナとその要件に基づいて、次のユースケースを特定します。

### アカウントマーケティングの最適化

マーケティングの影響を分析し、調査する：

- キャンペーン、チャネル、コンテンツをまたいで，
- アカウント内の購買グループ、セールスパイプラインの進行、アップセルおよびクロスセルの機会、顧客アカウントの健全性です。

[詳細なユースケースの記事](optimize-account-marketing.md)では、Customer Journey Analytics B2B editionがアカウントマーケティングの最適化にどのように役立つかを詳しく説明しています。

### 主要アカウントの拡大

主要アカウント内の購買グループをまたいで価値の高い顧客接点を特定し、事前のマーケティングおよび営業アクションを伝え、売上を増加させることができます。 アカウントレベルで顧客の生涯価値を計算し、監視。

[詳細なユースケースの記事](grow-key-accounts.md)では、Customer Journey Analytics B2B editionが主要アカウントの拡大にどのように役立つかを詳しく説明しています。

### 製品価値の構築

製品リリースと製品使用がアカウントおよびユーザーレベルで顧客満足度に与える影響を測定。 そのため、製品の特徴や機能を最適化し、今後の製品開発に情報を提供することができます。

[詳細なユースケースの記事](build-product-value.md)では、Customer Journey Analytics B2B editionが商品の価値を構築するのにどのように役立つかについて説明しています。


## 設定

これらのユースケースをサポートするようにCustomer Journey Analytics B2B editionを設定する方法については、[設定の記事](setup.md)を参照してください。
