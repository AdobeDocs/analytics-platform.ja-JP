## アドバンスフィルターオプション

フリーフォームテーブルのディメンションの隣にあるフィルターアイコンをクリックし、「アドバンスを表示」をクリックすると、次の条件を使用してフィルタリングできます。

* 次を含む
* doesNotContain
* すべての語句を含む
* いずれかの語句を含む
* フレーズを含む
* いずれの語句も含まない
* このフレーズを含まない
* equals
* 次と等しくない
* startsWith
* endsWith
