# Customer Journey AnalyticsとAdobe Analyticsのデータフィードの比較

Customer Journey AnalyticsとAdobe Analyticsの両方のデータフィードでは、生データをサードパーティのプラットフォームに書き出すことができます。 以前にAdobe Analyticsでデータフィードを使用した場合は、次の情報を使用して、使用可能な機能やデータ処理動作などの違いを把握します。
