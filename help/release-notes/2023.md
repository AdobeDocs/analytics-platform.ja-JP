# 2023年 Customer Journey Analytics リリースノート

## 2023年1月リリースノート {#jan23}

| 機能 | 説明 | [ロールアウトの開始](/help/release-notes/releases.md) | [一般公開](/help/release-notes/releases.md) |
| ----------- | ---------- | ----- | --- |
| **プロファイルおよびルックアップデータセットでのオブジェクト配列のサポート** | プロファイルデータセットとルックアップデータセットでは、CJA でのオブジェクト配列の使用がサポートされるようになりました。 | 2023年1月11日（PT） | 2023年1月19日（PT） |
| **ワークスペースのフォルダー** | フォルダーを使用すると、プロジェクトを整理および分類して、取得やアクセスを改善できます。さらに、共有された&#x200B;**[!UICONTROL 会社]**&#x200B;フォルダーを使用すると、管理者は、コンテンツを容易に作成してすべてのワークスペースユーザーと共有できます。[詳細情報](https://experienceleague.adobe.com/docs/analytics/analyze/analysis-workspace/build-workspace-project/workspace-folders/about-folders.html?lang=ja) | 該当なし | 2023年1月11日（PT） |
| **デフォルトのランディングページ** | 今年初めに導入された[新しいランディングページ](/help/getting-started/landing.md)は&#x200B;**2023年1月11日（PT）**&#x200B;にすべてのユーザーにとってデフォルトのエクスペリエンスとなる予定です。従来のランディングページは廃止される予定で、すべてのユーザーは新しいエクスペリエンスを利用する必要があります。 | 該当なし | 2023年1月11日（PT） |
| **プロジェクトマネージャーページを廃止** | 新しいランディングページのリリースに伴い、**[!UICONTROL Customer Journey Analytics]**／**[!UICONTROL コンポーネント]**&#x200B;の下に表示されるように、**[!UICONTROL プロジェクトマネージャー]**&#x200B;を廃止しました。新しいランディングページには、古いプロジェクトマネージャーページのすべての機能が含まれています。[詳細情報](https://experienceleague.adobe.com/docs/analytics-platform/using/cja-overview/landing.html?lang=ja#deprecate-pm-page) | 該当なし | 2023年1月11日（PT） |
| **Report Builder でのワークブックのスケジュール** | Customer Journey Analytics では、一定の間隔でワークブックを送信するスケジュールを作成できます。受信者は、ワークブックの最新の更新を定期的に受け取ることができるようになりました。[詳細情報](https://experienceleague.adobe.com/docs/analytics-platform/using/cja-reportbuilder/schedule-reportbuilder.html?lang=ja) | 該当なし | 2023年1月11日（PT） |
| **新しいプロジェクトの自動保存** | Analysis Workspace は新しく作成したプロジェクトを自動保存するようになりました。何らかの理由で、新しく作成したプロジェクトへのアクセス権が、手動で保存する前に予期せず失われた場合は、プロジェクトの復元バージョンを使用できるようになりました。以前は、プロジェクトは最初に手動で保存した後にのみ自動保存されていました。[詳細情報](/help/analysis-workspace/build-workspace-project/save-projects.md) | 該当なし | 2023年1月11日（PT） |
| **ユーザー環境設定を強化** | ユーザーレベル（[!UICONTROL コンポーネント]／[!UICONTROL 環境設定]）で追加の環境設定を指定できるようになりました。ユーザー環境設定を指定すると、選択内容はプロジェクト、テーブルおよびビジュアライゼーションをまたいで適用されます。環境設定ページには、次の新しいタブが含まれ、それぞれに新しい設定オプションが多数含まれています。<ul><li>フリーフォームテーブル</li><li>ビジュアライゼーション&lt;/li></ul>。さらに、「**[!UICONTROL 一般]**」タブと「**[!UICONTROL プロジェクト]**」タブに環境設定が追加されました。<p>以前は、これらの環境設定の多くは、個別のプロジェクト、テーブルおよびビジュアライゼーションに対してしか設定できませんでした。[詳細情報](/help/analysis-workspace/user-preferences.md) | 該当なし | 2023年1月11日（PT） |

| **IP からジオロケーションへのマッピングの改善** | 2022年9月29日（PT） | アドビの IP 検索ベンダーである Digital Element は、IP からジオロケーションへのマッピング用に新しく改善されたデータセット（NetAcuity Pulse）へのアップグレードを行います。Adobe Analytics では、この新しいデータセットの導入を **2023年1月11日（PT）**&#x200B;まで延期しました。新しいデータベースは、以前のバージョンよりも正確になります。新しいデータベースを採用する際に、IP からジオロケーションへのマッピングは一部変更または改善されます。<p> [!UICONTROL Analytics ソースコネクタ]を通じて提供される CJA データも、新しいマッピングを自動的に活用します。 |
